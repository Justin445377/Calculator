
# Calcolux

**Vibe Coding Test**

Calcolux ist ein moderner Taschenrechner, entwickelt als Testprojekt für Vibe Coding. Das Projekt demonstriert grundlegende Webentwicklung mit HTML, CSS und JavaScript.

## Features

- Grundrechenarten (Addition, Subtraktion, Multiplikation, Division)
- Vorzeichenwechsel (±)
- Prozentrechnung (%)
- Quadratwurzel (√)
- Sinus, Cosinus, Tangens (Winkel in Grad)
- Löschen (C), Rückschritt (⌫)
- Klammern
- Easter Egg (Antwort 42)
- Schwarz-Gold-Design mit visuellen Effekten
- Favicon und GitHub-Link mit Logo

## Schritte der Entwicklung

1. **Projektstart**: Initialisierung des Projekts als Vibe Coding Test.
2. **HTML-Struktur**: Erstellung der index.html mit allen Buttons und Anzeige.
3. **CSS-Design**: Gestaltung im Schwarz-Gold-Stil, Hinzufügen von Animationen und Effekten.
4. **JavaScript-Funktionen**: Implementierung der Grundfunktionen, Vorzeichenwechsel, Prozent, Quadratwurzel, trigonometrische Funktionen, Easter Egg.
5. **Favicon**: Erstellung und Einbindung eines SVG-Favicons.
6. **GitHub-Link**: Hinzufügen eines Links zum eigenen GitHub-Profil mit Logo.
7. **Dokumentation**: Beschreibung und Dokumentation aller Schritte in dieser README.

## Nutzung

Öffne die index.html im Browser. Alle Funktionen sind direkt nutzbar. Der Footer enthält einen Link zum GitHub-Profil.
